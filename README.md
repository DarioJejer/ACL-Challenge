

Technology Stack Selection:
    Backend: Go (Golang) 
    Framework: Gin
    Database: PostgreSQL
    ORM: GORM
    Testing tools: testify, mockery


Go (Golang): Chosen for its simplicity, strongly typed struckture, concurrency capabilities, and high performance in backend services.
Gin: Lightweight and fast HTTP framework that simplifies routing and middleware handling.
PostgreSQL: Reliable, open-source relational database with strong consistency.
GORN: It is the most popular Go ORM so it properly maintained by the community.
Testify: Provides expressive assertions and testing utilities, improving test readability and maintainability.
Mockery: Simplifies mock generation, enabling clean and isolated unit testing.
